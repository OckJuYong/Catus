# Catus 사용자 가이드

## 📱 Catus란?

Catus는 AI와 대화하며 일기를 작성하는 감정 기록 애플리케이션입니다.
일상의 이야기를 AI와 나누면, AI가 당신의 감정을 분석하고 아름다운 일기로 정리해줍니다.

## 🌟 주요 기능

### 1. AI와 대화하는 일기 작성
- 친근한 AI와 자유롭게 대화하세요
- 하루 동안 있었던 일을 자연스럽게 이야기하면 됩니다
- AI가 감정을 분석하고 요약해드립니다

### 2. 감정 기반 캘린더
- 매일의 감정이 색으로 표현됩니다
  - 🟢 행복 (초록색)
  - 🔵 슬픔 (파란색)
  - 🟡 불안 (노란색)
  - 🔴 화남 (빨간색)
  - ⚪ 보통 (회색)
- 한 눈에 한 달의 감정 흐름을 확인하세요

### 3. 익명 응원 메시지
- 다른 사용자에게 익명으로 응원 메시지를 보낼 수 있습니다
- 힘든 시기에 따뜻한 위로를 받아보세요

### 4. 다크 모드
- 눈이 편한 다크 모드를 지원합니다
- 설정에서 언제든 변경 가능합니다

## 🚀 시작하기

### 1단계: 로그인
1. 첫 화면에서 **카카오 로그인** 버튼을 클릭하세요
2. 카카오 계정으로 간편하게 로그인할 수 있습니다

### 2단계: 온보딩
1. 닉네임을 설정하세요
2. 간단한 질문에 답변하여 개인화된 경험을 받으세요
3. 튜토리얼을 통해 앱 사용법을 익히세요

### 3단계: 첫 일기 작성
1. **채팅** 버튼을 눌러 AI와 대화를 시작하세요
2. "오늘 하루 어땠어?"라는 질문에 자유롭게 답변하세요
3. AI가 계속해서 질문하며 대화를 이어갑니다
4. 대화가 끝나면 자동으로 그림일기가 생성됩니다

## 📖 화면별 사용법

### 🏠 홈 화면
- **최근 일기**: 최근에 작성한 일기를 확인하세요 (일기알람 시간이 되어 일기가 생성됐을 때 알람)
- **오늘의 감정**: 오늘의 기분을 한눈에 볼 수 있습니다

### 💬 채팅 화면
- AI와 대화하며 일기를 작성하는 공간입니다

**사용 팁:**
- 솔직하게 이야기하세요. AI는 판단하지 않습니다
- 간단한 대답도 괜찮아요. "좋았어", "힘들었어" 같은 짧은 답변도 OK
- 대화가 충분하다고 느껴지면 **대화 종료** 버튼을 누르세요
- 종료하면 AI가 감정을 분석하고 일기를 저장합니다

**대화 예시:**
```
AI: 안녕! 오늘 하루는 어땠어?
나: 친구들이랑 카페 갔다왔어

AI: 좋았겠다! 어떤 이야기를 나눴어?
나: 그냥 수다 떨면서 놀았어. 오랜만이라 재밌었어

AI: 오랜만에 만난 친구들이면 더 반가웠겠네! 다음에도 만날 계획이 있어?
나: 다음 주에 또 보기로 했어!
```

### 📅 캘린더 화면
- 월별로 일기를 확인할 수 있습니다
- 날짜를 클릭하면 그날의 일기를 볼 수 있습니다
- 감정별 색상으로 한 달의 감정 패턴을 한눈에 파악하세요

**기능:**
- 좌우 스와이프로 월 변경
- 날짜 클릭으로 일기 상세 보기
- 일기가 없는 날은 회색으로 표시

### 📝 일기 상세 화면
특정 날짜의 일기를 자세히 볼 수 있습니다.

**포함 내용:**
- 날짜 및 요일
- 감정 (이모지와 색상)
- AI가 작성한 일기 요약
- 당시 나눴던 대화 내역
- 수정/삭제 기능

**기능:**
- **수정하기**: 일기 내용을 직접 수정할 수 있습니다
- **삭제하기**: 일기를 삭제할 수 있습니다 (복구 불가)
- **공유하기**: 일기를 이미지로 저장하거나 공유할 수 있습니다

### 💌 응원 메시지 화면
익명으로 따뜻한 응원을 주고받는 공간입니다.

**받은 메시지:**
- 다른 사용자가 보낸 익명 응원 메시지를 확인하세요
- 읽지 않은 메시지는 강조 표시됩니다

**메시지 보내기:**
- 랜덤한 사용자에게 응원 메시지를 보낼 수 있습니다
- 완전히 익명이므로 누가 보냈는지 알 수 없습니다
- 긍정적이고 따뜻한 메시지를 보내주세요

**응원 메시지 예시:**
- "오늘도 수고했어요! 내일도 화이팅!"
- "힘든 일이 있어도 곧 좋아질 거예요"
- "당신은 충분히 잘하고 있어요"

### ⚙️ 설정 화면
앱의 다양한 설정을 관리할 수 있습니다.

**설정 항목:**
- **다크 모드**: 라이트/다크 테마 전환
- **알림 설정**: 푸시 알림 허용/차단
- **프로필 수정**: 닉네임 및 프로필 변경
- **개인정보 처리방침**: 개인정보 보호 정책 확인
- **로그아웃**: 계정에서 로그아웃
- **회원 탈퇴**: 계정 삭제 (모든 데이터 삭제)

## 💡 사용 팁

### 일기 작성 팁
1. **꾸준히 기록하세요**: 매일 짧게라도 기록하면 감정 패턴을 파악할 수 있습니다
2. **솔직하게 표현하세요**: AI는 판단하지 않습니다. 솔직한 감정을 표현하세요
3. **구체적으로 이야기하세요**: "좋았어"보다는 "친구와 맛있는 저녁을 먹어서 좋았어"가 더 좋습니다
4. **감정에 집중하세요**: 사건보다는 그때의 감정에 초점을 맞춰보세요

### 캘린더 활용 팁
- **월말 회고**: 매월 말에 캘린더를 보며 한 달을 돌아보세요
- **패턴 발견**: 어떤 요일에 특정 감정이 많은지 확인해보세요
- **감정 일기**: 감정별로 필터링하여 특정 감정이 든 날들을 모아보세요

### 응원 메시지 활용 팁
- **힘든 날 위로받기**: 슬프거나 힘든 감정의 일기를 작성한 날, 응원 메시지를 확인해보세요
- **나눔의 실천**: 좋은 일이 있었던 날, 다른 사람에게 응원을 보내보세요
- **긍정적 표현**: 비판이나 조언보다는 순수한 응원이 좋습니다

## 🔐 개인정보 보호

Catus는 당신의 프라이버시를 소중히 여깁니다.

- **암호화**: 모든 일기는 암호화되어 저장됩니다
- **익명성**: 응원 메시지는 완전히 익명으로 전달됩니다
- **데이터 관리**: 회원 탈퇴 시 모든 데이터가 삭제됩니다
- **외부 공유 금지**: 당신의 일기는 절대 외부에 공유되지 않습니다

## ❓ 자주 묻는 질문 (FAQ)

### Q1. 일기를 수정할 수 있나요?
A. 네! 일기 상세 화면에서 **수정하기** 버튼을 눌러 언제든 수정할 수 있습니다.

### Q2. 하루에 여러 개의 일기를 쓸 수 있나요?
A. 현재는 하루에 하나의 일기만 작성할 수 있습니다. 기존 일기를 수정하거나 새로 대화를 시작하면 이전 내용이 대체됩니다.

### Q3. AI가 분석한 감정이 잘못된 것 같아요
A. 일기 수정 시 감정을 직접 변경할 수 있습니다. AI의 감정 분석은 참고용이며, 최종적으로는 사용자가 선택합니다.

### Q4. 응원 메시지는 누가 보낸 건가요?
A. 완전히 익명이므로 알 수 없습니다. 이는 순수한 응원을 주고받기 위한 의도적인 설계입니다.

### Q5. 카카오 로그인만 지원하나요?
A. 현재는 카카오 로그인만 지원합니다. 향후 다른 소셜 로그인도 추가될 예정입니다.

### Q6. 일기를 백업할 수 있나요?
A. 현재는 서버에 자동으로 저장됩니다. 향후 로컬 백업 기능이 추가될 예정입니다.

### Q7. 오프라인에서도 사용할 수 있나요?
A. 현재는 인터넷 연결이 필요합니다. PWA 기능을 통한 오프라인 지원이 계획되어 있습니다.

### Q8. 일기를 다른 사람이 볼 수 있나요?
A. 아니요. 당신의 일기는 오직 당신만 볼 수 있습니다. 응원 메시지도 익명으로 전달되어 일기 내용은 공유되지 않습니다.

## 🐛 문제 해결

### 로그인이 안 돼요
1. 카카오 계정 상태를 확인하세요
2. 브라우저 캐시를 삭제해보세요
3. 다른 브라우저로 시도해보세요
4. 팝업 차단이 해제되어 있는지 확인하세요

### AI 응답이 느려요
1. 인터넷 연결 상태를 확인하세요
2. 잠시 후 다시 시도해보세요
3. 페이지를 새로고침해보세요

### 일기가 저장되지 않아요
1. 대화를 충분히 나눈 후 **대화 종료**를 눌러주세요
2. 인터넷 연결을 확인하세요
3. 로그인 상태를 확인하세요

### 앱이 느려요
1. 브라우저 캐시를 삭제해보세요
2. 다른 탭을 닫아보세요
3. 브라우저를 재시작해보세요
